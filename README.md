I build small, inspectable AI research systems around one question:

**Can model-driven improvement be measured without fooling ourselves?**

Current focus:

- [`nanoWM`](https://github.com/AlexandreRobicquet/nanoWM): world models from tiny gridworlds to transformers, RSSM, MuZero, and JEPA.
- [`loom`](https://github.com/AlexandreRobicquet/loom): budgeted orchestration experiments for critique, selection, and model comparison.
- [`nanorlm`](https://github.com/AlexandreRobicquet/nanorlm): recursive language-model experiments with explicit memory policies.

My bias: runnable code, fixed budgets, pre-registered rubrics, and artifact-backed claims.
